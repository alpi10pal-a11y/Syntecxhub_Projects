# Data Science Learning

This repository contains my learning projects in Data Science.  
Currently includes **Project 1: Iris Dataset Analysis**.

## 📂 Project Structure
- `Project1.ipynb` → Jupyter Notebook with analysis and visualizations  
- `Iris.csv` → Raw dataset used for training and testing  
- `iris.numpy.npy` → Preprocessed NumPy array for faster computation  
- `.ipynb_checkpoints` → Auto-saved notebook checkpoints  
- `README.md` → Project documentation  

## 🚀 Project 1: Iris Dataset
The Iris dataset is a classic dataset used for classification tasks.  
In this project, I:
- Explored the dataset using Pandas and NumPy  
- Visualized data distributions with Matplotlib/Seaborn  
- Implemented basic classification models  

## 🛠️ Technologies Used

- Pandas, NumPy  
- Matplotlib, Seaborn
